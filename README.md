This project is a simple game of rock, paper, scissors against a computer player, named Kiki. The player's moves are detected through computer vision. 
This project uses python libraries like opencv and mediapipe to handle hand tracking and gesture recognition.

Python version: 3.12
Mediapipe version: 0.10.14
Opencv version: 4.13.0.92

In order to run this program, you need to:
1- Clone this repo
2- Download opencv-python and mediapipe
3- Ensure that you are using the appropriate versions of the libraries as well as the python interpreter

Hope you enjoy playing with Kiki! :D
